# Battleship-AH Steam Scope

## Product Identity

Battleship-AH is the polished first public game built on AlienHand.

It is the product that introduces:

- private local multiplayer stations
- host-held hidden state
- one host PC serving more than one player view

## In-Scope For First Release

- two-player Battleship
- one host PC
- one local host view
- one LAN PC endpoint for the second private station
- keyboard and gamepad support
- pairing flow
- private ship placement
- private turn-by-turn combat
- transition/privacy screens
- end-of-match flow

## Out-Of-Scope For First Release

- Smart TV endpoints
- multi-mouse input
- more than two players
- online matchmaking
- account systems
- cosmetic progression
- platform-generalized launcher experience

## Quality Bar

The first release must prioritize:

- very clear onboarding
- low setup confusion
- visually legible private/public state separation
- clean pacing between turns
- premium-feeling UI and audio feedback

## Minimum Steam-Worthy Standard

Before release, the game should have:

- stable end-to-end pairing and play
- clear recoverability from disconnect or mis-assignment
- obvious player station identity
- enough presentation quality to feel intentional, not prototype-like
