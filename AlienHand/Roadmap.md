# AlienHand Roadmap

## Stage A - Hidden-State Proof

Deliver Battleship-AH as the polished proof of:

- host authority
- private stations
- LAN endpoint projection
- station-specific views

## Stage B - Input Novelty Proof

Follow with the two-hand mouse-controlled bullet-hell title as the proof of:

- multi-mouse identity
- dual-hand control language
- AlienHand as an input innovation platform

## Stage C - Generalized Endpoint And Input Expansion

Expand into:

- richer station composition
- additional endpoint classes
- raw multi-device support
- asymmetric and mixed-role games

## Stage D - Virtual Hand Interface Research

Pursue:

- virtual hand abstraction
- multi-mouse keyboard overlays
- summonable desktop interaction layers
- eventual custom hardware

This stage must remain outside the critical path for the first Steam release.
