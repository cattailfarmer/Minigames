# AlienHand Games

AlienHand Games is a host-mediated local multiplayer platform that turns one PC into a multi-station game console.

Core idea:

- one host machine acts as the authoritative processor
- multiple stations can be composed from displays, endpoints, and input devices
- stations can expose private or shared game state
- games run on the host while views and control channels are assigned per station

Tagline:

AlienHand turns one PC into a true local multiplayer console - multiple screens, multiple players, multiple minds controlling one machine.

Initial development focus:

1. host-authoritative session model
2. station and endpoint model
3. pairing and assignment flows
4. LAN PC endpoint before Smart TV endpoints
5. Battleship-AH as the pilot proof-of-platform game
6. reusable media tooling for audio, VFX, tracker composition, and analysis
